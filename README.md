# Assignment 2 – Interactive Portfolio Website

This project is an improved version of my personal portfolio website from Assignment 1.  
In this version, I focused on making the website more interactive, modern, and user-friendly using JavaScript, dynamic behavior, animations, and theme switching.

## Features

- Dark / Light mode toggle
- Theme preference saved using localStorage
- Dynamic project filtering by category
- Time-based greeting message
- Contact form validation with clear user feedback
- Error messages for invalid input
- Success message after valid form submission
- Smooth hover effects and reveal animations
- Responsive layout for different screen sizes
